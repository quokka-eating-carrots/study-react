# study-react

## webpack과 babel 설치하기
```
$ npm i -D webpack-cli
$ npm i -D webpack-dev-server
$ npm i -D babel-loader
$ npm i -D @babel/core @babel/preset-env @babel/preset-react
$ npm i -D html-webpack-plugin
```
